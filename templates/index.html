Welcome to IOT Device Management

API Endpoints:

  • Create a Device: 
    POST /api/devices/

    Send a JSON payload with 'uid' and 'name' to create a new device.

  • Delete a Device: 
    DELETE /api/devices/{device-uid}

    Replace '{device-uid}' with the UID of the device you want to delete.

  • Retrieve a Device: 
    GET /api/devices/{device-uid}

    Replace '{device-uid}' with the UID of the device you want to retrieve.

  • List all Devices: 
    GET /api/devices/

    Retrieve a JSON list of all devices.

  • Get Readings for a Device: 
    GET /api/devices/{device-uid}/readings/{parameter}/?start_on=yyyy-mm-ddTHH:MM:SS&end_on=yyyy-mm-ddTHH:MM:SS

    Replace '{device-uid}' with the UID of the device, '{parameter}' with 'temperature' or 'humidity', and provide 'start_on' and 'end_on' query parameters for date filtering.
